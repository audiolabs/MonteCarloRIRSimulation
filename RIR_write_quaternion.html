RIR_write_quaternion module

RIR_write_quaternion.generate_rir(Azimuth, Elevation, source_array_dist, Room, ReverberationTime, dist, num_mics, array_type)

Function that generates ‘M’ room impulse responses for a given ‘DOA’. Array and source are randomly positionend and rotated in a room using quaternions.

Parameters:

  • Azimuth – (float) Azimuth angle of the source to the array center in rad

  • Elevation – (float) Elevation angle of the source to the array center in rad

  • source_array_dist – (float) distance of source signal to array center

  • Room – (array) x, y, z dimension of room

  • ReverberationTime – (float) Reverberation time of the current room

  • dist – (float) intermicrophone distance for ULA or radius for CUA

  • num_mics – (int) number of microphones

  • array_type – (str) one of ULA, CUA, SUA

Returns:

Tuple of a unit-norm vector pointing from array to source and an array of room impulse responses

Return type:

Tuple

RIR_write_quaternion.generate_source_vec(sad, Az, El)

Function that generates a source vector with given length and given Azimuth and Elevation w.r.t. array center The rotation is performed using quaternions

Parameters:

  • sad (float) – Distance between source and arraycenter

  • Az (float) – Azimuth angle in rad

  • El (float) – Elevation angle in rad

Returns:

Vector which points from array center to source

Return type:

float array

RIR_write_quaternion.main()

This function gets input parameter from RIR_parameter.py and computes RIRs from them

Parameters:

  • roomx (str) – x position of room

  • roomy (str) – y position of room

  • roomz (str) – z position of room

  • j (str) – source array distance

  • ReverberationTime (str) – T60 of the room

  • dist (str) – inter-microphone distance (ULA) or radius (CUA, ..)

  • path (str) – path so save

  • array_type (str) – selects the type of array

  • num_mics (str) – Number of microphones

  • indx (str) – How often to go through these parameters based on parameter file … just used to change the saving name

RIR_write_quaternion.sample_room(room, wdist=1.0)

Function that samples a square room uniformly. The sample has to be within a certain distance to the wall

Parameters:

  • room – (array) x,y,z room dimensions

  • wdist – (float) minimal distance to walls of the sampling point

Returns:

(array) x,y,z sampled room position

Return type:

float array

RIR_write_quaternion.verify_DOA(mics, sourcevecoff, doa)
